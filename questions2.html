Group Two Content Questions

What are programming controls and design structures?
  • Controls
    • Specific user interface elements used to interact with code
  • Design structures
    • If statements
    • For loops
    • Functions
    • Classes
What are website testing procedures?
  • Functional testing
    • Ensure website works as intended from a user point of view
  • Performance testing
    • Ensure website runs quickly on all devices
  • Security testing
    • Make sure the website is secure against cyber attackers
  • Usability testing
    • Ensure that the website is intuitive and adheres to design requirements
What are website debugging methods?
  • Breakpoints
    • Stop code at specific spots to check variable values and identify issues
  • Debug log
    • Log errors and exceptions into a console
  • Code review
    • Sharing code with team members to help identify issues
What are website coding techniques?
  • Rapid Application Development (RAD)
    • Quick prototyping
    • Constant client feedback
    • Iteration
  • Waterfall
    • Sequentially complete each stage of a process one after another
  • Agile
    • Prototype major features and iterate with client before moving on to the next major feature
    • Similar to RAD
