Group Four Content Questions

The three major programming control structures.
  • Sequential
    • Execute instructions in a linear manner, from top to bottom.
  • Conditional
    • Involves decision making, such as if-else conditions
  • Iterative structure
    • Repeating instructions over and over, such as for loops or while statements
Hypertext markup language (HTML) and markup languages.
  • HTML
    • Method to construct a web page
    • Consists of tags, which define an element, alongside a value, which configures the element.
  • Markup languages
    • HTML is a markup language
    • Markup languages are used to create an interface for a website using tags
Cascading style sheets (CSS).
  • Allows users to define styles
    • Fonts
    • Colours
    • Organization
  • Enhance the visual appearance of a website
    • Enhance HTML documents
Syntax and uses of programming languages.
  • Syntax
    • Set of rules that define the way that variables, conditions, and other programming elements should be formatted
    • Essential for organization
    • Essential for program to function (e.g., { }, ;, ( ))
  • Uses of programming languages
    • Software development
    • Website development
    • Data analysis
    • Scripting
    • Artificial intelligence and machine learning
