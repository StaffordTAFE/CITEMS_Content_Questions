Group Three Content Questions

Authentication and web security.
  • Authentication
    • Identify the identity of a user
    • Username and password
    • Two-factor authentication (2FA)
  • Web security
    • Protocols implemented to protect against cyber-attacks
    • Authentication is vital for ensuring user information is not stolen
Hypertext transfer protocol (HTTP).

Hypertext transfer protocol (HTTP) is a method used to share information across the web. It defines how data should be packaged and formatted.

Session management.

Session management is the process of maintaining a specific state of a website. Certain information only exists over the duration of the session and is removed when a new session is started. Some information must be saved (user preferences, shopping cart items, login details) so it can be reused in future sessions.

Stateless programming.

Stateless programming is a method of formatting requests to a server to execute a task without the need for knowledge about previous requests. This makes the program more robust and easily scalable.
