Group One Content Questions

What are the principles of analysis and design?
  • Emphasis
  • Balance
  • Alignment
  • Contrast
  • Repetition
  • Proportion
  • Movement
  • White space
What are website architectural requirements?
  • Scalability
    • Website must be able to handle traffic
  • User interface
    • Must be compatible with multiple devices
    • Must effectively implement principles of analysis and design
  • User experience
    • Must be accessible
    • Must be usable
  • Compliance and regulations
    • Must adhere to local laws
    • Must adhere to copyright laws
    • Must adhere to industry-specific laws
What are website design structures, including hierarchy and navigation design?

Website design structure is how a website is structured to enhance user experience (UX)

Hierarchy:

  • Header:
    • Outlines navigation and core website information
  • Sections and categories
    • Organize similar content into categories
  • Typography
    • Use fonts and font sizes to structure hierarchical content
  • Emphasis
    • Use color, contrast, and/or size to structure elements hierarchically

Navigation Design:

  • Headers
    • Primary form of navigation
    • Clearly shows most of the pages within a website
  • Dropdown menus
    • Compact information
    • Reduces clutter
  • Mobile friendly
    • Buttons are big enough to be pressed with a finger
What are user-interface design requirements and production processes?
  • Simplicity
    • Simple design
    • Clear language
  • Consistency
    • Ensure that design style stays consistent throughout the website
  • Accessibility
    • Website must be accessible for people with learning disabilities, visual or auditory impairment
    • Website must be accessible for multiple languages, cultures, and religions
  • Responsive design
    • Must adapt to different screen sizes
    • Must work on mobile devices
